# Volatility-Consumer-Sentiment-Analysis
Analyzing the relationship between market volatility and consumer sentiment using FRED data and COVID-19 policy controls.
## Data Sources
- **Oxford COVID-19 Government Response Tracker (OxCGRT):** The raw policy stringency data used in this project is sourced from the official Oxford University repository.
  [Download Raw Oxford Dataset](https://raw.githubusercontent.com/OxCGRT/covid-policy-dataset/main/data/OxCGRT_compact_national_v1.csv)
